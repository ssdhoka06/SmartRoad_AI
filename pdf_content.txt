🧠  Ragini  — Build rl_environment.py (Gymnasium Custom Env)  
1. Create rl_environment.py in your project folder. Import gymnasium, numpy, and import the pipeline 
(SegFormer + YOLO outputs) from pipeline.py.  
2. Define class DriverEnv(gymnasium.Env). Set metadata = {'render_modes': ['human']}.  
3. Define self.observation_space using gymnasium.spaces.Box: shape should be a flat vector of ~10 
floats — phone_detected (0/1), cigarette_detected (0/1), driver_face_detected (0/1), gaze_away (0/1), 
phone_near_face (0/1), phone_duration (float 0–30), gaze_duration (float 0–30), seg_driver_zone (0/1), 
seg_steering_visible (0/1), activity_count (int 0–10). Dtype: np.float32.  
4. Define self.action_space using gymnasium.spaces.Discrete(3): 0 = All Clear, 1 = Monitor, 2 = Violation.  
5. Implement reset(self, seed=None): reset all internal counters (phone_duration=0, gaze_duration=0, 
step_count=0), return (obs_vector, info_dict).  
6. Implement step(self, action): call pipeline to get latest frame detections → build obs vector → compute 
reward using: +1 if action==2 and real violation, -1 if action==2 and no violation (false alert), -2 if 
action!=2 and real violation occurred. Return (obs, reward, terminated, truncated, info).  
7. Implement _get_obs(self): reads the detection results from YOLO (phone, cigarette, face) and seg 
outputs (driver zone, steering zone), builds and returns the numpy float32 obs vector.  
8. Add a __main__ block: env = DriverEnv(); obs, _ = env.reset(); print(obs). Run it and confirm no errors. 
Share output screenshot with team.  
Output ✓  rl_environment.py — runnable Gymnasium env with correct obs/action spaces

3. Implement compute_attention_score(tracker) function that takes a DurationTracker instance and 
returns a float 0–100.  
4. Write a simple test: simulate 90 frames (~3 seconds) of phone detected, compute score, print result.
